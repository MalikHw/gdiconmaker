FAQ

General Site Info

What is the primary purpose of this Icon Maker?

It converts any image you upload into a Geometry Dash icon disguised as an icon pack. You pick your picture, choose which gamemodes to generate (cube, wave, ball), give the pack a name, and download a ZIP that drops straight into your GD install. No Photoshop, no manual resizing, no reading wikis about texture atlases at midnight.

Is this tool free to use for the community?

Yes. No paywalls, no premium tiers(besides that april fools joke)), no "basic plan." The source code is on GitHub under GPL-v3. It will stay free. If you want to support the dev there is a donate link in the footer, but that's entirely optional.

Do I need to create an account to save my icon designs?

No account needed. Everything runs in your browser. Your author name gets saved to localStorage so you don't retype it every session, but there's no login, no email, no database with your data in it.

Is this site affiliated with RobTop Games?

No. This is a fan-made tool with no official connection to RobTop Games or Geometry Dash. It's a community project. Use it at your own discretion, same as any third-party GD tool.

How can I report a bug or suggest a new feature?

Open an issue on the GitHub Issues page for bugs. For feature requests, there's a Google Form linked in the footer. Please only discord if you wanna be friends, but i take those from the links here,

Icon Customization & Mechanics

Which Geometry Dash versions are currently supported?

GEODE: The tool targets GD 2.2 (specifically 2.2081) with Geode's Texture Loader. If you're on an older version, the pack won't even show up - Manual file replacement: in the zip data and u swap files manually? yes (ONLY 2.2 THO)

How do I change the primary and secondary colors of my icon?

As of the February 15 2026 update, you no longer need to manually set player colors to match your icon. Just install the pack and GD handles the colors automatically.

Are glow effects and special gradients available?

The tool itself doesn't add effects on top — what you put in is what you get out, scaled and formatted correctly. its a IMAGE to icon

Can I preview animations or just static poses?

Static only for now. The generated pack is a set of still textures. GIF inputs are supported but only the first frame gets used(hard for animated), so you're not getting animated icons in-game from this tool(yet...).

Exporting & Compatibility

In what file formats can I download my finished icons?

You get a ZIP containing PNG files, structured exactly the way Geode Texture Loader expects. You don't need to unpack anything(unless no geode) — just use the ZIP as-is with Texture Loader.

Can I import these icons directly into my Geometry Dash game files?

Yes, two ways. The easier one: install Geode, grab the Texture Loader and More Icons mods from the index, then drag the ZIP into Settings > Graphics > Textures the same way Minecraft texture packs work. The manual way: open the ZIP and copy the icons folder into your GD resources folder, usually at C:\Program Files (x86)\Steam\steamapps\common\Geometry Dash\resources\. To revert, run a Steam integrity check on the game.

Are the exported images high-resolution (HD) or standard (SD)?

The tool generates HD and UHD output. The exact resolution scales based on your input image quality, so feeding it a low-res picture will still produce low-res icons — garbage in, garbage out.

Technical Troubleshooting

Does this tool work on mobile devices and tablets?

It does, but the crop interface is awkward on smaller screens. The site detects mobile and adjusts the layout. The actual pack generation works fine — it's just the cropping step that's a bit fiddly with touch input. but icons themselves for gd? Yes! as long as you have Geode

How do I clear my cache if the icons are not loading properly?

In Chrome or Edge: open DevTools (F12), right-click the refresh button, select "Empty Cache and Hard Reload." In Firefox: Ctrl+Shift+Delete, then clear cached web content. If the site has a service worker cached from a previous version, that can also cause stale behavior — you can unregister it from DevTools under Application > Service Workers.

Does the site support custom Texture Packs?

The output this tool generates is a texture pack, formatted for the Texture Loader mod. You're not applying someone else's pack — you're building your own from an image. If you want to combine this pack with another existing texture pack, Just drag the pack made from here to the top in Texture Loader, or just use the More Icons mod
